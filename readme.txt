TrueMoney Payment Gateway service by TMPAY.NET.


== Description ==

XTMPAY GATEWAY คือระบบตัดบัตร TrueMoney  ซึ่งให้บริการโดย tmpay.net.

Major features in TMPAY GATEWAY include:

* TMPAY Setting ตั้งค่าระบบพื้นฐาน
* TrueMoney Transaction List ระบบจัดเก็บรายการบัตร ที่สมาชิกทำรายการ เพื่อดูประวัติการทำรายการต่างๆ



PS: ยังไม่มีอะไรใหม่.



== Installation ==

Upload TMPAY plugin ขึ้นเว็บไซต์, จากนั้นกด Activate, เรียบร้อย ระบบพร้อมใช้งาน!

PS: ถ้ากด Deactivate plugin ระบบจะทำการลบข้อมูลบัตร TrueMoney ทั้งหมด. หากต้องการลงใหม่ กด Activate plugin
